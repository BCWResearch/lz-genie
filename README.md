# layerzero-quickstart
A set of boilerplate, scaffolding and ready to use tools to get a dev up and running quickly on l0 usecases

# Getting Started

## Prerequisites

- Node.js > 18.20.0
- yarn

```bash
npm install -g yarn
```

## Installation

```bash
yarn install
```

## Run Locally

```bash
yarn install:local
npx lzgenie
```

## Development

```bash
yarn start
```
